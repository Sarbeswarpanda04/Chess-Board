# ♟️ Pixel-Perfect CSS Grid Chessboard

A beautifully structured, responsive, and pixel-perfect standard $8 \times 8$ chessboard built using modern HTML5 semantic elements and advanced CSS Grid/Flexbox layouts. Complete with perfectly aligned file (A–H) and rank (1–8) alphanumeric border coordinates that never overlap or clip piece assets.

![Chessboard Preview](https://github.com/Sarbeswarpanda04/Chess-Board/blob/main/chessboardpreview.png?raw=true)

---

## 🚀 Features

* **Advanced Layout Architecture:** Leverages a nested sub-grid structure within a parent $3 \times 3$ CSS Grid to keep coordinates perfectly isolated from the board content.
* **Zero Asset Overlap:** Grid padding and layout tracks prevent labels from bleeding into the squares, ensuring chess piece graphics remain perfectly centered and completely visible.
* **Pure CSS Styling:** No heavy JavaScript calculation overhead—rendering is entirely native, lightweight, and lightning-fast.
* **Responsive Sizing Constraints:** Built using controlled grid-fraction parameters to preserve proportions beautifully across different display resolutions.

---

## 🛠️ Built With

* **HTML5:** Semantic markup structure separating coordinate layouts from game-board nodes.
* **CSS3:** Advanced Grid layouts, flexbox centering engines, and clean modern styling variables.
